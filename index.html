SISTEMA DE GESTIÓN ISHAJON

Automatización de Flujos de Trabajo de RRHH

1. DESCRIPCIÓN DEL PROYECTO

Aplicación web integral diseñada para digitalizar el proceso de solicitud, aprobación y seguimiento de permisos laborales. La solución reemplaza los formularios físicos con un flujo de trabajo electrónico estructurado que garantiza trazabilidad y eficiencia.

2. ARQUITECTURA TÉCNICA

  • Backend: Construido con PHP 8.1 y Framework Laravel 11.x. Implementación robusta de Eloquent ORM para el modelado de datos y uso de Policies para la gestión granular de seguridad.
  • Base de Datos: PostgreSQL 17.4. Esquema relacional optimizado con 8 tablas principales, índices para búsquedas rápidas y claves foráneas para integridad referencial.
  • Frontend: Arquitectura reactiva utilizando Blade Templates potenciado con Livewire. Esto permite interacciones dinámicas (SPA feel) sin la complejidad de frameworks JS externos, facilitando el mantenimiento.

3. LÓGICA DE NEGOCIO

  • Flujo Jerárquico: Sistema de aprobación multinivel (Empleado -> Director -> RRHH). Cada nivel posee permisos específicos de escritura y validación.
  • Seguridad: Autenticación robusta, protección CSRF/XSS y sistema de Firma Digital criptográfica para validar cada aprobación.
  • Notificaciones: Motor de eventos que dispara correos electrónicos transaccionales en cada cambio de estado de la solicitud.

ARQUITECTURA & SRS

Especificación de Requerimientos de Software

Documentación técnica que cierra la brecha entre las necesidades del negocio y la implementación del código. Este enfoque garantiza escalabilidad y mantenibilidad.

REQ-01: GESTIÓN DE ACCESO (RBAC)

Objetivo: Implementar un sistema de control de acceso basado en roles para asegurar la integridad de los datos.

  • Implementación de Middleware de autorización en rutas críticas.
  • Encriptación de contraseñas utilizando Bcrypt.
  • Sesiones seguras con tiempo de expiración configurable.

REQ-02: TRAZABILIDAD Y AUDITORÍA

Objetivo: Cada acción crítica (aprobación/rechazo) debe ser inmutable y rastreable.

  • Registro de logs detallados (Usuario, IP, Timestamp, Acción).
  • Firmado digital de registros para evitar repudio.

class RBAC_Middleware {

    if (! $request->user()->hasRole($role)) {
      return response()->json(['error' => 'Unauthorized'], 403);
    }
    return $next($request);
  }

HERRAMIENTAS SAP POS

Automatización y Soporte Operativo Nivel 2

1. SOPORTE TÉCNICO ESPECIALIZADO

Gestión de incidentes complejos en el punto de venta (POS). Diagnóstico de fallos en la comunicación con el ERP central, resolución de errores de replicación de datos (IDOCs) y soporte a hardware periférico.

2. AUTOMATIZACIÓN (SCRIPTING)

Desarrollo de scripts en Python y consultas SQL avanzadas para automatizar tareas repetitivas:

  • Limpieza de logs antiguos para optimizar rendimiento.
  • Extracción masiva de reportes de ventas locales en casos de caída de red.
  • Verificación de integridad de base de datos local.

3. CAPACITACIÓN

Liderazgo en procesos de formación a usuarios finales (cajeros y supervisores) para asegurar el uso correcto de las nuevas funcionalidades del sistema.
