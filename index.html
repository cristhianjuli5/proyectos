SISTEMA DE GESTIÓN ISHAJON

Automatización de Flujos de Trabajo de RRHH

1. DESCRIPCIÓN DEL PROYECTO

Aplicación web integral diseñada para digitalizar el proceso de solicitud, aprobación y seguimiento de permisos laborales. La solución reemplaza los formularios físicos con un flujo de trabajo electrónico estructurado que garantiza trazabilidad y eficiencia.

2. ARQUITECTURA TÉCNICA

  • Backend: Construido con PHP 8.1 y Framework Laravel 11.x. Implementación robusta de Eloquent ORM para el modelado de datos y uso de Policies para la gestión granular de seguridad.
  • Base de Datos: PostgreSQL 17.4. Esquema relacional optimizado con 8 tablas principales, índices para búsquedas rápidas y claves foráneas para integridad referencial.
  • Frontend: Arquitectura reactiva utilizando Blade Templates potenciado con Livewire. Esto permite interacciones dinámicas (SPA feel) sin la complejidad de frameworks JS externos, facilitando el mantenimiento.

BOT DE AUDITORÍA

Automatización de Procesos de Infraestructura

1. PROPÓSITO

Eliminar la carga manual de revisar logs de servidores diariamente. Este bot se ejecuta cada madrugada, analiza archivos de registro en busca de patrones de error (Regex) y genera alertas.

2. FLUJO DE TRABAJO

  • Conexión SSH: Itera sobre una lista de inventario de servidores.
  • Análisis: Parsea `/var/log/syslog` buscando palabras clave ("Error", "Fail", "Panic").
  • Reporte: Compila un resumen CSV y lo envía por correo electrónico a la gerencia TI.
